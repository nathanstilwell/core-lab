            font-weight : bold;
          }

Class Selector

My favorite, stay tuned to find out way


<!-- HTML -->
           <div class="my-div">blah blah blah </div>

/* CSS */
          .my-div {
            font-size : 100px;
            font-weight : bold;
          }

Id selector


<!-- HTML -->
        <div id="my-div">blah blah blah </div>

/* CSS */

          }

Combinators

Because you probably want more than one thing

Descentant Selector

Stuff inside of stuff


<!-- HTML -->
          <div class="my-div">
            <p>
              some stuff
            </p>
          </div>

/* CSS */
          .my-div p{
            color : purple;
          }

Adjacent sibling

The + combinator matches the second element only if it is immediately following the first element


<!-- HTML -->
        <div class="my-div">
          <p class="test-p">
            some stuff
          </p>

          <p>
            some other stuff
          </p>
        </div>

/* CSS */
        .my-div .test-p + p{
          color : purple;
        }

General sibling

The ~ combinator separates two selectors and matches the second element only if it is preceded by the first, and both share a common parent.


<!-- HTML -->
          <div class="my-div">
            <p class="test-p">
              some stuff
            </p>

            <p>
              some other stuff
            </p>


            <p>
              yet, even more stuff
            </p>
          </div>

/* CSS */
          .my-div .test-p ~ p{
            color : purple;
          }

Child Selector

The > combinator separates two selectors and matches only those elements matched by the second selector that are direct children of elements matched by the first.


<!-- HTML -->
          <div class="my-div">
            <p>
              some stuff
            </p>

            <p>
              some other stuff
            </p>

            <p>
              yet, even more stuff
            </p>
          </div>

/* CSS */
          .my-div > p{
            color : purple;
          }

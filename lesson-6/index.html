Lesson 6

Background

Background Color

Sets the background of the element to a solid color.

Available Values: Any valid color, or none

Background Color

Background Color 

  .example {
    background-color: hotpink;
  }

Background Image

Sets the background of an element to an image

Available Values: URL to any valid image, or none

Background Image

Background Image

  .example {
    background-image: url(../img/rose.png);
  }

Background Repeat

Sets repeat value for background image

Available Values: repeat, no-repeat, none

Background Repeat

Background Repeat
Background Repeat

  .example {
    background-image: url(../img/leaf.svg);
    background-repeat: no-repeat;
  }
    

  .example {
    background-image: url(../img/leaf.svg);
    background-repeat: repeat;
  }

Background Attachment

Sets how background should behave during scroll

Available Values: scroll, fixed

Background Attachment

Background Attachment
Background Attachment

  .example {
    background-image: url(../img/floral.png);
    background-attachment: scroll;
  }
    

  .example {
    background-image: url(../img/floral.png);
    background-attachment: fixed;
  }

Background Position

Sets the position of the background relative to the element

Available Values: top, center, bottom; left, center, right; Pixels

Background Position

Background Position
Background Position

  .example {
    background-image: url(../img/ninja.png);
    background-position: top right;
  }
    

  .example {
    background-image: url(../img/ninja.png);
    background-position: bottom left;
  }

Background Size

Sets how the size of the background image relative to the element

Available Values: Percentage, Pixels, cover, contain

Background Size

Background Size

  .example {
    background-image: url(../img/booze.jpg);
    background-size: auto 75%;
  }
